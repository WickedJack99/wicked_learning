# Concept

Learning content should be highly interactive.

Long passive reading phases should be avoided where possible.

Content should be interrupted by small reflection prompts, retrieval questions and self-explanation tasks.

The purpose of these interactions is not assessment but learning reinforcement.

Formal tests and competency evaluations should exist, but they should not be the primary learning mechanism.

User progress should be tracked, like last time module completed, maybe just updated at of a counter of times completed.
Users are allowed to redo tests and evaluations as often as they want.
Track correct incorrect answer per question count for statistics where to optimize, add another module for the topic, rework the module.
Suggest rework of modules related to question. Store topics related to question with weight which is between 1 and 0.1 for calculation / priority which modules are more related / more likely to rework.

Stuff where learning happens is checked by ai if  suitable for learning, checked if too long reading phases, too long passive time spent by learners, long videos avoided, all according to actual science.
