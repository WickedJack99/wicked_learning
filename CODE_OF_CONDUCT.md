# Code of Conduct

## Our Pledge

This project is meant to be thoughtful, exploratory and welcoming. We want
contributors and users to feel respected while building a platform that supports
meaningful learning.

We pledge to make participation in this project a harassment-free experience
for everyone, regardless of age, body size, disability, ethnicity, gender
identity and expression, level of experience, education, nationality, personal
appearance, race, religion or sexual identity and orientation.

## Expected Behavior

Examples of behavior that helps this community:

- using welcoming and inclusive language
- respecting different viewpoints and experiences
- giving constructive feedback with care
- accepting responsibility and apologizing when mistakes happen
- focusing on what is best for the project and its users
- assuming good intent while still addressing real harm

## Unacceptable Behavior

Examples of behavior that is not acceptable:

- harassment, intimidation or discriminatory comments
- personal attacks or sustained disruption
- sexualized language or imagery in project spaces
- publishing private information without explicit permission
- dismissing accessibility, privacy or safety concerns without discussion
- conduct that would reasonably be considered inappropriate in a professional
  open-source setting

## Scope

This Code of Conduct applies in project spaces such as issues, pull requests,
discussions, documentation, chat spaces and any public or private interaction
where someone represents the project.

## Enforcement

Project maintainers are responsible for clarifying expected behavior and may
take appropriate action when this Code of Conduct is violated. Actions may
include a private warning, temporary restrictions, removal from project spaces or
other measures needed to protect the community.

If you need to report a concern, contact the project maintainers through the
repository's issue or contact channels. If a public issue would expose private
information or create additional harm, use a private contact method instead.

## Attribution

This Code of Conduct is inspired by common open-source community standards,
including the Contributor Covenant, and is adapted for this project.
