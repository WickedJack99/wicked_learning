# Researching and Developing an open-source SDT-based learning platform

> Exploring how psychology, AI, and game design can make learning something people genuinely want to do.

## Why?

Many digital learning platforms rely on points, badges, streaks, and leaderboards to motivate learners.

While these mechanics can increase engagement, I became interested in a different question:

> What if people wanted to learn even without external rewards?

This project is an experiment in designing a learning environment around intrinsic motivation instead of extrinsic incentives.

It is heavily inspired by research in Self-Determination Theory (SDT), Cognitive Load Theory, Multimedia Learning, and self-regulated learning.

The goal is not simply to build another learning platform.

The goal is to explore how software can support curiosity, mastery, autonomy, and meaningful progress.

---

## What is this repository?

This repository documents both the ideas and the implementation behind the project.

You'll find:

- architecture decisions
- design concepts
- research notes
- implementation progress
- things that worked
- things that didn't

Rather than presenting a finished product, this repository documents the journey.

The goal is to openly share ideas, experiments, and design decisions as they evolve. I hope they encourage thought, discussion, and perhaps inspire others on their own learning journey.

---

## Current areas of exploration

- Intrinsic motivation (Self-Determination Theory)
- Meaningful gamification
- AI-assisted learning
- Knowledge management
- Browser extension integration
- Scenario-based learning
- Long-term knowledge retention
- Community-driven learning

## Status

🚧 Early development.
