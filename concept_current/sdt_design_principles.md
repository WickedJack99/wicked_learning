# SDT Design Principles

## Avoid

- Global points.
- Competitive leaderboards.
- Public ranking.
- Streak pressure.
- Status badges as the main motivator.
- Feedback that shames learners.
- Artificial engagement loops that distract from learning.

## Prefer

- Informational feedback.
- Optional exploration.
- Personal progress reflection.
- Learner-owned notes.
- Retry paths that feel normal.
- Unlocks that represent understanding rather than reward extraction.
- Cooperative features without popularity contests.

## MVP Interpretation

The current dashboard shows completed activity state only as personal orientation. The question feedback explains the reasoning and offers a review loop when the selected answer misses the key clue.
